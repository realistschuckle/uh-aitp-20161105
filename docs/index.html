Curly braces...

Group related statements together.

The curly braces belong to the code immediately proceeding them.

Here, the curly braces belong to the if statement. Everything inside the open and close curly braces will only run if the condition for the if statement evaluates to true 


if (someNumber < 10)
{
    // All of these statements will only run
    // if "someNumber" is less than 10.
    Console.WriteLine("Yes, you have less than 10.");
    Console.WriteLine("Don't worry. That's more than most.");
}

Here, the curly braces belong to the namespace. Everything inside the namespace's curly braces belong to the namespace. In this case, the MyAwesomeClass definition will exist in the My.Awesome.Program namespace. 


namespace My.Awesome.Program
{
    public class MyAwesomeClass
    {
        // Some awesome class stuff in here
    }
}

Here, the curly braces belong to the method Main. All of the statements inside the curly braces will execute when some other code executes the Main method. 


public static void Main(string[] args)
{
    Pizza myPizza = new Pizza("large", "new york");
    Console.WriteLine("Your pizza will cost: " + myPizza.Cost);
}

In this code, when some other code runs Main, then a new Pizza gets created and we print its cost.
